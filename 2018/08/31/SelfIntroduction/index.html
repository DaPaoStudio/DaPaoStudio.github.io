Self Introduction | 自我介绍与评价

我叫张伟，来自西安电子科技大学计算机科学与技术学院，Gap两年后，今年大三。
​ 我从小热爱游戏，游戏也塑造了我。大学时选择了软件工程专业并在校以策划身份参与了数款游戏Demo的制作，参加过三次Game Jam和三次Hackathon，还参加过西山居游戏的暑期实习和腾讯游戏策划公开课，曾担过微软学生俱乐部活动部部长和软件学院大学生科技协会游戏组组长，并以创始人兼初代社长身份在一年半的时间经营壮大了西安地区第一个游戏开发社团：Nova独游社。我喜欢游戏和计算机技术，将脑中有趣的想法实现使我快乐，成为优秀的制作人并通过游戏给玩家带来快乐是我的毕生追求，未来的目标是提高代码能力的同时进一步学习、实践和总结游戏设计方法论以期获得全流程独立开发能力。

  • 对工作拥有很强的责任心，做事善始善终，组织和沟通能力强，在校期间创建并运营了游戏开发社团并组织参与多款游戏的开发均完成了作品，策划公开课的作品在提交后仍在保持迭代
  • 拥有较强的学习能力、逻辑思维与系统分析能力，对热门游戏会尝试分析拆解
  • 了解Unity引擎，拥有良好的代码功底，也熟悉各种工具的使用，具备一定的落地能力
  • 热爱游戏，游戏对我而言就是那个可以终身投入进去的宏大而有说服力的故事，能以设计者的身份参与到游戏行业的进步将使我倍感幸福

Project Experience |项目经历

XidianDefender

《西电塔防》是大一下学期时使用Unity引擎复刻的一款以本校为主要场景元素的《保卫萝卜》换皮塔防游戏

主要工作：完成了大部分代码的编写和游戏内像素帧动画的绘制

Saloon Arena 客厅竞技场

《客厅竞技场》是我在Global Game Jam 2019 上海虹口站和几名随机组队的队友一起做的Game Jam项目。GJ主题为“What home means to you”，游戏以四人同屏收集垃圾获得分数作为核心玩法，在此基础上引入电量、被撞击和攻击会损失电量、电量耗尽复活前会被贴近的玩家掠夺身上的垃圾等机制，又限制了地图大小和场景中的垃圾总量外加四种精心设计的风格迥异的角色，让玩家可以灵活选择游戏进程中的战术，也可以玩得很欢乐。

主要工作：前期参与破题、玩法设计、角色设计，在中期负责了手感调校、后期负责匹配游戏里的音乐和音效。

A.L.I.C.E 化身孤岛的鲸

《A.L.I.C.E》是19年4月为了清明集训和中国计算机设计大赛做的一个第一人称鲸鱼模拟器，游戏以最孤独的鲸鱼Alice和“鲸落”、“搁浅”，以游戏作为媒介向玩家讲述了一个孤独而凄美的故事。

主要工作：游戏玩法设计、角色设计、简单的地形编辑、故事框架编写、统筹、宣传、音乐音效匹配。

诛心5

《诛心5》是在CiGA Game Jam 2019 西安站完成的一款Game Jam作品。游戏核心玩法极其简单，躲避屏幕中按预定模式飞行的❤并手刃掉面前飞来的❤，随着游戏时长的推移❤的同屏数量和行为模式会越来越复杂，因为游戏高度致敬了早期街机游戏且打击感不错，外加游戏保留了可以记录姓名的分数排行榜，在试玩阶段前去试玩打榜的玩家络绎不绝。

主要工作：破题、游戏核心玩法设计、游戏音乐音效匹配、操作感打击感调校

破碎境界

《破碎镜界》是2019年暑期在西山居SEED训练营实习时和同小组的同事一起做的以“镜像”为题的MiniGame。这是一款故事风格偏向黑童话的纵版清关Roguelike游戏。

主要工作：核心玩法设计、Roguelike系统设计、武器设计、道具设计、敌人设计，后期负责了游戏里的音校匹配

这个项目凭借着精致的画面、完整的机制、爽快的战斗体验和有深度的剧情在最后的评选中斩获了最佳创意奖和最优作品奖。

代号：新星

《代号：新星》是大三上学期末在社团内开始预研的一款Roguelike和STG相结合的手游，目前仍在制作中。在具备飞行射击游戏爽快体验的同时，也相当注重Roguelike的重玩价值。

  • [流派]游戏提前设计了多种流派，比如霰弹流、贴脸莽夫流、导弹海流。虽然我们是按照Build设计的武器和道具，但如果把不同流派的Key物品互相组合，也会产生意想不到的 效果

  • [多机种]游戏内置了数种主角战舰，同种战舰也有多套初始装配

  • [随机关卡]游戏通过一定算法生成每局都不同的关卡并在其中安插了丰富的随机事件

主要工作：游戏制作人，负责了游戏核心玩法设计、角色设计、Roguelike系统设计、关卡设计、敌人设计、武器设计、道具设计、程序美术需求文档撰写、项目管理

左右五千年

《左右五千年》是2021年腾讯游戏策划公开课中开发的一款以考古为主题的平台跳跃解谜游戏。

主要工作：系统设计、程序实现

这个项目凭借着精致的画面与极致的完成度获得了上海站的银奖。

Inter Experience | 实习经历

2019/07-2019/08 西山居SEED训练营 游戏策划实习生

在6周的时间内学习游戏策划的基础知识，学习游戏设计的相关理论和方法并在最后3周利用所学知识与同组的6名成员合作开发一款以“镜像”为主题的MiniGame

主要工作：创意阶段以”镜像”为主题通过头脑风暴和会议破解题目并确定“LOL辅助和射手般心意相通”的核心体验，根据游戏核心体验敲定了核心玩法。在Demo阶段完成了Roguelike系统的设计和游戏中Build的设计、Boss和道中小Boss设计，参与了关卡设计、敌人设计。还负责了与公司音频部门和美术部门的对接。

在最终的评定中，游戏获得了“最佳创意奖”、“最优作品奖”，自己被评为“优秀营员”
